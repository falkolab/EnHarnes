# Harness Feedback

dev policies, methodology docs) — not product bugs and not task failures.

[`harness.feedback`](../.claude/skills/harness.feedback/SKILL.md) skill. Append
newest entries at the bottom (append-only). Operator triages periodically and
flips each entry to `<-- TRIAGED: ...>`.
